Groupomania

Contexte (fictif) :

- Je suis développeur depuis plus d'un an chez CONNECT-E, une petite agence web regroupant une douzaine d'employés.

- Le client est Groupomania, un groupe spécialisé dans la grande distribution et l'un des plus fidèles clients de l'agence.

- Le projet consiste à développer le MVP d'un réseau social interne pour les employés de Groupomania. Le but de cet outil est de faciliter les interactions entre collègues.

Mission :

- Authentifier un utilisateur et maintenir sa session

- Gérer un stockage de données à l'aide de SQL

- Implémenter un stockage de données sécurisé en utilisant SQL

- Personnaliser le contenu envoyé à un client web

Back end :

- La base de données est créée et gérée grace à l'ORM Sequelize

- Les routes sont sécurisées, le back envoie un token d'authentification

- Mise en place de rôles administrateur et moderateur par défaut

- Géstion des fichiers avec "multer"

Front end :

- Le front est réalisé avec REACT et REDUX

- La charte des couleurs est en rapport avec le logo fourni

- L'utilisateur peut commenter ou "liker" les posts des autres utilisateurs

- L'utilisateur a la possibilité de modifier / supprimer ses données, posts et commentaires

Ce projet complet m'a beaucoup apporté.
Le développement complet d'une application en utilisant des outils modernes a été une tache certes difficile pour un débutant mais gratifiante sur le plan personnel.
Ce projet m'a permis d'aborder REACT, REDUX, l'ORM SEQUELIZE et d'approfondir mes bases en javascript, base de données etc.
